# Student Grade Tracker

## Project Description

Student Grade Tracker is a Java-based console application developed as part of the CodeAlpha Java Programming Internship.

This project helps in managing student academic records efficiently. It allows users to add student details, calculate grades, find average marks, highest and lowest scores, search student records, and generate a summary report.

## Features

* Add Student Details
* View All Students
* Calculate Average Marks
* Find Highest Marks
* Find Lowest Marks
* Search Student by Roll Number
* Summary Report
* Marks Validation (0–100)

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class
* Eclipse IDE

## How to Run

1. Open the project in Eclipse IDE.
2. Run `StudentGradeTracker.java`.
3. Use the menu options to manage student records.
